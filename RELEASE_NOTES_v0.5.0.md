## [v0.5.0] - 2026-04-29

### ✨ Features
- **ES:** Integración de la API Piped para búsquedas estables y sin restricciones en YouTube Music.
- **EN:** Integration of Piped API for stable and unrestricted searches on YouTube Music.
- **ES:** Nuevo "Plan C": Scraper HTML dinámico como método infalible de búsqueda.
- **EN:** New "Plan C": Dynamic HTML scraper as a fail-proof search method.
- **ES:** Rediseño Premium (Liquid UI) en la pantalla de Búsqueda con efectos de cristal esmerilado y fondos animados.
- **EN:** Premium Redesign (Liquid UI) on Search Screen with frosted glass effects and animated backgrounds.

### 🛠 Improvements
- **ES:** Sistema de redundancia multi-servidor y evasión de bloqueos Cloudflare (anti-bots).
- **EN:** Multi-server redundancy system and Cloudflare block evasion (anti-bots).

### 🐛 Fixes
- **ES:** Corrección de actualización de estado (`StateFlow`) que causaba pantallas en blanco. Error de sintaxis en el cliente Ktor resuelto.
- **EN:** Fixed state update (`StateFlow`) causing blank screens. Resolved syntax error in Ktor client.

### ⚙️ Tech
- **ES:** Transición del inestable cliente interno (InnerTube `WEB_REMIX`) a interfaces públicas e indexación web directa.
- **EN:** Transition from unstable internal client (InnerTube `WEB_REMIX`) to public interfaces and direct web indexing.

### 🚀 Release Notes (ES)
¡Bienvenidos a **FridaMusic v0.5.0**! 🎵 Esta actualización marca un hito en la estabilidad de la aplicación. Hemos reconstruido por completo el motor central que conecta con YouTube. Despedimos a los bloqueos de región y de API, y damos la bienvenida a un sistema de triple capa: API Piped, servidores espejo y un raspador web puro que garantiza que **siempre** encuentres tu música. Además, estrenamos un diseño espectacular que hará que explorar tu biblioteca sea un placer visual.

### 🚀 Release Notes (EN)
Welcome to **FridaMusic v0.5.0**! 🎵 This update marks a milestone in app stability. We have completely rebuilt the core engine connecting to YouTube. Say goodbye to region and API blocks, and welcome a triple-layer system: Piped API, mirror servers, and a pure web scraper ensuring you will **always** find your music. Additionally, we are debuting a spectacular design that will make exploring your library a visual delight.
